## Memory Acquisition

O propósito deste repositório é documentar meu estudo prático e teórico a respeito da aquisição de memória no contexto da análise forense digital, expondo conceitos fundamentais como a importância de preservar a validade e a utilidade forense das evidências obtidas durante o processo de aquisição, além de registrar experimentos práticos sobre procedimentos, ferramentas e técnicas utilizadas para realização da aquisição de memória em diferentes ambientes computacionais modernos.
